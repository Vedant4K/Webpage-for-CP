A Quick Look

Competitive Programming is a mental sport which enables you to code a given problem under provided constraints. Now, consider a programming contest as a game of cricket, metaphorically. Compile a code and submit, you may get a WA (Wrong Answer). Make changes to code and eventually you will get your first AC (Accepted/Correct Answer).
